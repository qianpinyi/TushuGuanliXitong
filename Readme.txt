文件结构：
DEV：使用的第三方库SDL2的相关文件
PAL_Library：自己写的库的相关源码
--PAL_BasicFunctions：基础功能/杂项相关
----PAL_BasicFunctions_0.cpp：不太好分类的一些功能
----PAL_BasicFunctions_Charset.cpp：Windows平台字符集转换相关函数的封装
----PAL_BasicFunctions_Color.cpp：颜色相关结构体和函数
----PAL_BasicFunctions_Config.cpp：配置文件管理类源码
----PAL_BasicFunctions_Debug.cpp：调试相关功能
----PAL_BasicFunctions_Posize.cpp：区间、点、矩形管理类
----PAL_BasicFunctions_StringEX.cpp：字符串功能拓展相关函数和类
--PAL_DataStructure：数据结构相关类
----PAL_SplayTree.cpp：封装成类的伸展树的源码
----PAL_TrieTree.cpp：封装成类的字典树的源码
--PAL_GUI/PAL_GUI_0.cpp：GUI库的源码（课设主体部分）
resources：课设程序用到的相关图片资源
msyhl.ttc：字体文件
*.dll第三方库的动态链接库文件
1.cpp：调用写的库做的程序的部分
1.exe：已生成的可执行文件
cfg.txt：程序相关设置的配置文件
Log.txt：默认调试日志输出
report.txt：课设报告的纯文本版
BookData.dat：书库信息文件
课设报告.pdf：课设报告
Readme.txt：本文件
